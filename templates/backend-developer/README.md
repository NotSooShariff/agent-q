# Backend Developer Agent

Expert in server-side logic, APIs, databases, and backend architecture.

## Overview

This agent is a senior backend developer who specializes in:
- RESTful and GraphQL API development
- Database design and optimization
- Authentication and authorization
- Microservices architecture
- Performance optimization and caching
- Security best practices

## When to Use

Invoke this agent when you need help with:
- Designing and implementing API endpoints
- Database schema design and migrations
- Server-side business logic
- Authentication flows (OAuth, JWT, sessions)
- Performance optimization and caching strategies
- Backend security and input validation

## Capabilities

- **API Development**: REST, GraphQL, gRPC, WebSocket
- **Databases**: SQL (PostgreSQL, MySQL), NoSQL (MongoDB, Redis)
- **Frameworks**: Express, FastAPI, Django, Spring Boot, Rails
- **Architecture**: Microservices, layered architecture, SOLID principles
- **Security**: Input validation, SQL injection prevention, API security
- **Performance**: Query optimization, caching, load balancing

## Example Usage

```
User: "Help me design a user authentication API with JWT"
Agent: [Provides comprehensive auth implementation with security best practices]

User: "Optimize this database query that's causing N+1 problems"
Agent: [Analyzes query, suggests eager loading or query optimization]

User: "Design a REST API for an e-commerce shopping cart"
Agent: [Designs endpoints, data models, and implementation strategy]
```

## Tech Stacks Supported

- Node.js (Express, NestJS, Fastify)
- Python (Django, FastAPI, Flask)
- Java (Spring Boot)
- Go (Gin, Echo)
- Ruby (Rails, Sinatra)
- PHP (Laravel, Symfony)
